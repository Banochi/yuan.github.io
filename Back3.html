Project Description

I created a periodic table using HTML and CSS to practice my web development skills and apply what I’ve learned about structuring content and styling it visually. Each element is displayed in a grid layout, with colors representing different element groups like metals, nonmetals, and noble gases. This project helped me better understand both the layout of the periodic table and how to use CSS for design and positioning. It was a fun way to combine science with coding.
