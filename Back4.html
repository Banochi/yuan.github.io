Project Description

I created an SDG sign-up page using only HTML and CSS. In this project, users can enter their name and choose their favorite Sustainable Development Goal from a dropdown list. The goal of the page is to promote awareness of the 17 SDGs while giving users a simple way to share which one matters most to them. This project helped me improve my skills in creating forms, using HTML elements properly, and styling everything with CSS to make it look clean and organized—even without using JavaScript.
